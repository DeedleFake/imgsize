Image Size

Welcome to Image Size. This is a simple web-app that fetches images from a URL and resizes them. The idea is to make it easier to embed images on forums that don't support resizing, or other similar situations. The app has support for PNG, JPEG, and GIF files, but the output is currently always PNG.

To resize an image to specific width or height with a preserved aspect ratio, simply leave the other dimension blank. For exmaple, setting width to 100 and leaving height blank will resize a 200x100 image to 100x50.

This app uses the resize package to preform the actual resizing.
